# Jawhara Jewellery — Gate Mall Branch
## Internal Management System

A luxury dark-mode web-based internal management system for the Jawhara Jewellery Gate Mall Branch. Built with pure HTML, CSS, and JavaScript with a RESTful Table API for data persistence.

---

## ✅ Completed Features

### 1. Login System
- Admin credentials: `username: jhr` / `password: 12345`
- Session-based authentication (sessionStorage)
- Password show/hide toggle
- Elegant login page with gold accent design

### 2. Dashboard Analytics
- **KPI Cards**: Today's Profit, Monthly Profit, Gold Profit, Diamond Profit (all in KWD)
- **Daily Performance Bar**: Visual progress toward 418 KWD daily target
- **Performance % Calculation**: `(Total Profit / 418) × 100`
- **Gold/Diamond Split**: Based on profit (not sales value)
  - `Gold % = (Gold Profit / Total Profit) × 100`
  - `Diamond % = (Diamond Profit / Total Profit) × 100`
- **Trend Chart**: Day-by-day line chart for Gold & Diamond profit (current month)
- **Doughnut Chart**: Category profit split visualization
- **Recent Sales Table**: Last 6 transactions at a glance

### 3. Daily Sales Entry
- Full form: Date, Category (Gold/Diamond), Item Name, Quantity, Sale Price, Cost Price, Notes
- Live profit calculator preview
- Validation and error feedback
- Edit existing records (form pre-fills automatically)

### 4. Sales Records (Full CRUD)
- Complete list of all sales with pagination (10 per page)
- Search by item name or notes
- Filter by category (Gold/Diamond)
- Filter by month
- Edit and Delete buttons for each record
- Confirmation modal before deletion

### 5. Monthly Printable Report
- Text-based header: "Jawhara Jewellery" / "The Gate Mall Branch" / "Monthly Sold Pieces Report"
- Auto-generated month & year
- Table: Date | Item | Category | Profit (KWD)
- Summary: Total Sold Items, Gold Profit, Diamond Profit, Total Profit
- Footer: "Generated by Sales System" + date + Manager Signature line
- A4 optimized print layout
- **PDF Export** via html2canvas + jsPDF
- Buttons hidden when printing

## 💾 Data Model

### Table: `sales`

| Field | Type | Description |
|-------|------|-------------|
| `id` | text | Auto-generated UUID |
| `date` | text | Sale date (YYYY-MM-DD) |
| `item_name` | text | Jewelry item name |
| `category` | text | Gold or Diamond |
| `quantity` | number | Units sold |
| `sale_price` | number | Sale price in KWD |
| `cost_price` | number | Cost/purchase price in KWD |
| `profit` | number | Calculated: (sale - cost) × qty |
| `notes` | text | Optional notes |

---

## ⚙️ Business Logic

- **Profit** = (Sale Price − Cost Price) × Quantity
- **Gold %** = (Gold Profit / Total Profit) × 100
- **Diamond %** = (Diamond Profit / Total Profit) × 100
- **Performance %** = (Today's Total Profit / 418) × 100
- **Daily Target**: 418.000 KWD
- **Currency**: KWD (3 decimal places)
